3.1 POINTS SUPPORT
DECIMAL PLACES

You can now use float numbers as points and list stats will sum them up (up to 2 decimal places).

3.2 LIST STATS SHOW
SUMMED UP TIME ENTRIES

Time entries are now summed up in list stats, provided that the input makes sense (hours and minutes).

3.3 CARD TITLE MARKUP
SUPPORTS _ITALIC_

Previously you could only use *text* and now you can use _text_ as well. Always good to have options.

3.4 CATEGORIES FILTER SUPPORTS
"UNGROUPED" OR "UNCATEGORIZED"

You can now filter out cards that do not have a group/category.

3.5 PRO4TRELLO FILTERS AND TRELLO FITLERS WORK TOGETHER NOW

You will no longer mess up Trello's label filtering if you are using Pro4Trello filters, any of them.

How can I contribute?

1. Review the extension

If you think Pro4Trello is awesome - you can leave a rating in the Chrome Store. This will help other people learn about the extension. Also make sure to leave your feedback or a feature request!

Do you have a blog, a video channel or write for tech. media? Consider reviewing Pro for Trello in a video or an article. The author is always available for a comment or any kind of feedback.

2. Make a pull request

If you can help you likely should, Pro4Trello is public on GitHub: github/mdjekic/pro4trello

3. Donate

If Pro for Trello makes your work or life easier please consider donating.
